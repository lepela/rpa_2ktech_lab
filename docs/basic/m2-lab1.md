# M2 실습 — 첫 번째 흐름 만들기

## 🎯 학습 목표

- PAD에서 새 흐름을 생성합니다.
- 액션을 추가하고 실행하는 기본 과정을 익힙니다.
- 변수 패널에서 값을 실시간으로 확인합니다.

## ⏱ 예상 소요 시간

| 구분 | 시간 |
|------|------|
| 실습 | 20분 |
| 확장 실습 | 5분 |

---

## 준비물

- Power Automate Desktop 설치 완료
- PAD 콘솔에 Microsoft 365 계정 로그인 상태

---

## 시나리오

현재 날짜와 시간을 가져와서 메시지 박스로 화면에 출력하는 흐름을 만듭니다.

완성 결과:

```
[메시지 박스]
현재 시간: 2024-01-15 오전 10:30:00
```

---

## 단계별 가이드

### 1단계. 새 흐름 생성

1. PAD 콘솔에서 **[+ 새 흐름]** 버튼 클릭
2. 흐름 이름 입력: `M2_날짜시간_출력`
3. **[만들기]** 클릭 → 디자이너 화면 열림

<!-- SCREENSHOT: 새 흐름 만들기 대화상자 -->

### 2단계. 현재 날짜 및 시간 가져오기 액션 추가

1. 왼쪽 액션 패널 검색창에 `날짜` 입력
2. **[날짜 및 시간] > [현재 날짜 및 시간 가져오기]** 더블클릭 (또는 캔버스로 드래그)
3. 액션 설정 창에서:
   - **검색 대상:** 현재 날짜 및 시간
   - **저장 변수:** `CurrentDateTime` (기본값 유지)
4. **[저장]** 클릭

<!-- SCREENSHOT: '현재 날짜 및 시간 가져오기' 액션 설정 창 -->

### 3단계. 메시지 표시 액션 추가

1. 액션 패널 검색창에 `메시지` 입력
2. **[메시지 박스] > [메시지 표시]** 더블클릭
3. 액션 설정 창에서:
   - **메시지:** `현재 시간: %CurrentDateTime%`
   - **메시지 박스 제목:** `결과 확인`
4. **[저장]** 클릭

{: .note }
> `%CurrentDateTime%` — 중괄호가 아니라 **퍼센트 기호**입니다. PAD에서는 `%변수명%`으로 변수를 참조합니다.

<!-- SCREENSHOT: '메시지 표시' 액션 설정 창 — %CurrentDateTime% 입력된 상태 -->

### 4단계. 실행 및 확인

1. 디자이너 상단의 **[▶ 실행]** 버튼 클릭
2. 메시지 박스가 화면에 나타나는지 확인
3. **오른쪽 변수 패널**에서 `CurrentDateTime` 변수 값 확인
4. 메시지 박스의 **[확인]** 클릭해서 닫기

<!-- SCREENSHOT: 실행 중 변수 패널에 CurrentDateTime 값이 표시된 화면 -->

## 확장 실습 — 메시지 커스터마이즈

아래 단계를 스스로 시도해보세요.

1. **[변수 설정]** 액션을 추가해서 텍스트 변수 `%인사말%`에 `"안녕하세요!"` 저장
2. 메시지 표시 액션의 내용을 `%인사말% 현재 시간: %CurrentDateTime%`으로 수정
3. 실행 후 결과 확인

---

## 💡 LLM 활용 팁

날짜 형식을 바꾸고 싶다면 LLM에게 물어보세요.

**질문 예시:**
```
PAD에서 CurrentDateTime을 "yyyy년 MM월 dd일" 형식의 
텍스트로 변환하는 표현식을 알려줘
```

**기대 답변 형태:**
```
%FormatDateTime(CurrentDateTime, "yyyy년 MM월 dd일")%
```

---

## ✅ 핵심 정리

- 흐름 생성 → 액션 추가 → 실행의 기본 흐름을 완주했습니다.
- 변수 패널에서 실행 중 값을 실시간으로 확인할 수 있습니다.
- `%변수명%`으로 액션 설정 안에서 변수를 참조합니다.
