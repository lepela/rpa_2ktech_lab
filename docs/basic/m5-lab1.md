# M5 실습 — 템플릿 흐름

## 🎯 학습 목표

- PA 템플릿으로 클라우드 흐름을 빠르게 만들어봅니다.
- 흐름 테스트 방법을 익힙니다.
- "이런 것도 자동화가 된다"는 체험으로 Day 2 기대감을 형성합니다.

## ⏱ 예상 소요 시간

| 구분 | 시간 |
|------|------|
| 실습 | 15분 |
| 버퍼 | 5분 |

---

## 준비물

- `make.powerautomate.com` 로그인 상태
- Microsoft Teams 접근 가능

---

## 시나리오

버튼 하나 누르면 내 이메일로 오늘의 날씨 알림을 보내는 인스턴트 흐름을 만들어봅니다.  
(날씨 커넥터가 없는 환경이면 간단한 알림 메시지로 대체합니다.)

---

## 단계별 가이드

### 1단계. make.powerautomate.com 접속

1. 브라우저에서 `make.powerautomate.com` 접속
2. 왼쪽 메뉴에서 **[내 흐름]** 클릭
3. **[+ 새 흐름] > [인스턴트 클라우드 흐름]** 클릭

<!-- SCREENSHOT: make.powerautomate.com 새 흐름 선택 화면 -->

### 2단계. 트리거 선택

1. 흐름 이름: `M5_알림버튼`
2. 트리거 선택: **"수동으로 흐름 트리거"**
3. **[만들기]** 클릭

### 3단계. 이메일 발송 액션 추가

1. 디자이너에서 **[+ 새 단계]** 클릭
2. 검색창에 `이메일 보내기` 입력
3. **[Office 365 Outlook] > [이메일 보내기(V2)]** 선택
4. 설정:
   - **받는 사람:** 본인 이메일
   - **제목:** `[자동화 테스트] 알림이 도착했습니다`
   - **본문:** `PA 인스턴트 흐름이 정상 동작하고 있습니다. 실행 시각: [트리거 타임스탬프 동적 콘텐츠 추가]`

<!-- SCREENSHOT: PA 디자이너 — 이메일 액션 설정 화면, 동적 콘텐츠 메뉴 열린 상태 -->

{: .note }
> **동적 콘텐츠 추가 방법:**  
> 본문 입력창 클릭 → 오른쪽에 나타나는 **[동적 콘텐츠 추가]** → `타임스탬프` 선택  
> PAD의 `%CurrentDateTime%`과 같은 역할입니다.

### 4단계. 저장 및 테스트

1. **[저장]** 클릭
2. **[테스트]** 버튼 클릭 → **[수동]** 선택 → **[테스트]**
3. **[흐름 실행]** 클릭
4. 이메일 수신 확인
