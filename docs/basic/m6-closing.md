## 오늘 배운 것 — 한눈에 정리

| 모듈 | 핵심 내용 |
|------|----------|
| **M0. 오리엔테이션** | 환경 확인, 학습 철학 (개념 최소 / LLM 활용 / 직접 체험) |
| **M1. Power Platform** | PA vs PAD 차이, 테넌트/환경, Attended vs Unattended |

| **M3. 웹+엑셀** | 셀렉터 개념, 자동캡처 한계, CSS 셀렉터 정제, 엑셀 읽기/쓰기 |
| **M3-Lab** | 오피넷 유가 → 엑셀 누적 기록 — 서브흐름 첫 적용 |

| **M5. PA 클라우드** | 인스턴트/자동화/예약 흐름, 커넥터, PAD↔PA 연결 구조 |
| **M5-Lab** | 템플릿 흐름 체험 |
